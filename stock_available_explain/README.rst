=======================
Stock available explain
=======================

This module shows why the stock availability has a certain level in a certain
context.

Usage
=====

This module adds a wizard named `Explain Product Available` that can be
called to show how the (virtual) availability of a product is calculated in
general, or for certain locations and/or dates.
